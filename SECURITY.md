## APK Verification

All released APKs include checksums for verification:
```bash
# Verify MD5
md5sum echo-latest.apk

# Verify SHA256
sha256sum echo-latest.apk
```

Compare with checksums in the release notes.

## Reporting Security Issues

**For Echo app security issues:**
Contact the original developers at their repository.

**For mirror security issues:**
Open a private security advisory on this repository.
